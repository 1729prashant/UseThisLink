Run Redirection locally
------------------------
PORT=3000 go run cmd/main.go


Adding New URLs
-----------------
UNIX
-----
To test adding a new URL via the /shorten endpoint, you can use cURL with the POST method.

curl -X POST http://localhost:3000/shorten \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com"}'

OR curl -X POST http://localhost:3000/shorten -d '{"url": "https://example.com"}' 


Expected response:
{
  "short_url": "abc123"
}



WIN
-----
curl -X POST http://localhost:3000/shorten ^
  -H "Content-Type: application/json" ^
  -d "{\"url\": \"https://example.com\"}"

Expected response:
{
  "short_url": "abc123"
}



Redirect URLs
-----------------
UNIX
-----
curl -i http://localhost:3000/abc123

Expected Response:
You should see a 302 Found HTTP status code and a Location header pointing to the original URL (e.g., https://example.com), like this:
HTTP/1.1 302 Found
Location: https://example.com
...


WIN
-----
curl -i http://localhost:3000/abc123

Expected Response:
Similar to UNIX
